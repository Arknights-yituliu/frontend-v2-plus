## 设计配色方案（基于原主题色）

以原主题色 Primary `#1867C0`（蓝）为核心，统一 Vuetify、Naive UI 与自定义样式，保持项目一贯的观感与对比度。

### 颜色规范
- Light
  - Primary `#1867C0`，OnPrimary `#FFFFFF`，PrimaryDarken `#1F5592`
  - Secondary `#48A9A6`，OnSecondary `#FFFFFF`
  - Accent/Hover `#60ACFF`
  - Background `#FFFFFF`，Surface `#FFFFFF`，SurfaceVariant `#F5F7FB`
  - Outline `#D7DFE8`
  - Success `#4CAF50`，Warning `#FB8C00`，Error `#B00020`，Info `#2196F3`
- Dark
  - Primary `#1867C0`，PrimaryDarken `#1F5592`
  - Secondary `#48A9A6`
  - Accent/Hover `#60ACFF`
  - Background `#141414`，Surface `#222222`，SurfaceVariant `#323232`
  - Outline `#4A5568`
  - Success `#4CAF50`，Warning `#FB8C00`，Error `#B00020`，Info `#2196F3`
- Brand Gradient（侧栏/装饰）
  - 方案A（当前生效）：`#1867C0` → `#1F5592`（主色到深主色）

### 集成指引
- Vuetify 主题（推荐覆盖内置 `light`/`dark` 名称）
  - 编辑 `src/plugins/vuetify/vuetify.js`：
    - 用以上颜色分别定义 `light` 与 `dark` 主题（键：`background`/`surface`/`primary`/`secondary`/`success` 等）。
    - 保持 `theme.global.name.value = 'light'|'dark'` 的切换逻辑不变。
- Naive UI
  - 在 `src/App.vue` 的 `themeOverrides` 中，将 `common.primaryColor` 等同步至 Primary/Accent。
- CSS 变量
  - 在 `src/assets/css/common/theme.scss` 中维护：
    - `--brand-gradient-start`、`--brand-gradient-end`（当前已设为 `#667eea → #764ba2`）
    - 现有 `--c-*` 变量与主题保持一致（无需变更即可生效）。
  - 侧栏背景已改为变量驱动：
    `.navigation-drawer { background: linear-gradient(135deg, var(--brand-gradient-start), var(--brand-gradient-end)) !important; }`

### 使用示例
- Vue 组件（Vuetify 色值）
  ```vue
  <v-btn color="primary">主要操作</v-btn>
  <v-alert type="success" variant="tonal" />
  ```
- 自定义样式（CSS 变量）
  ```scss
  .card {
    background: var(--c-card-background-color);
    border-color: var(--c-border-color);
  }
  ```
