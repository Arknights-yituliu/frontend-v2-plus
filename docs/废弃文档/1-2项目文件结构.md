## 项目结构
```
├─docs 文档目录
│ README.md  
│ vite.config.js  vite配置文件   
├─src 源码
│   ├─api    封装的API的js文件目录，将API抽象复用
│   ├─assets 资源文件
│   │  └─css  样式文件
│   │      ├─layout 网站布局
│   │      ├─sprite 雪碧图
│   │      ├─material 材料模块
│   │      ├─tool 工具模块
│   │      └─survey 调查模块
│   ├─components 布局组件
│   ├─custom 自定义组件
│   │  └─css 
│   ├─pages 页面目录
│   │  ├─about  网站信息页面目录
│   │  ├─dev 部分开发中页面目录
│   │  ├─material 材料相关页面目录
│   │  ├─survey 干员调查页面目录
│   │  └─tools 实用工具页面目录
│   ├─static 静态文件目录
│   │   └─json 一般是放一些数据的json格式文件
│   ├─utils 自定义工具函数
```
