# API

## 请求格式示例

### HTTP GET 


|  名称   | 说明                              |
|:-----:|:--------------------------------|
| 请求URL | 	域名/API路径?参数名=参数值&参数名=参数值...... |


### HTTP POST:

|   名称   | 说明                                                              |
|:------:|:----------------------------------------------------------------|
| 请求 URL | 	域名/API路径                                                       |
|  请求体   | 	请求体可以使用 JSON 也可以使用 Form 表单, 需要注意的是, 请求的 Content-Type 是一定要设置准确的 |

HTTP POST JSON 格式:

```
{
    "参数名": "参数值",
    "参数名2": "参数值"
}
```

HTTP POST 表单格式:

```
param1=value&param2=value
```
